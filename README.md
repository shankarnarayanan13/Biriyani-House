🍗 Biriyani House Website

A simple and attractive restaurant landing page website built using HTML and CSS.
This project showcases different biriyani varieties and food items with a modern UI design.

---

📸 Website Preview

"Biriyani Website" (Screenshots/biriyani.png)

---

🚀 Features

- Responsive restaurant landing page
- Navigation bar with multiple sections
- Hero section with attractive biriyani banner
- Food varieties showcase
- Clean and simple UI design
- Organized assets folder for images

---

🛠️ Technologies Used

- HTML5
- CSS3
